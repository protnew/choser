# Handover Document: Project Choser (Modular API & Research Agent)

Этот документ предназначен для синхронизации работы различных ИИ-агентов над проектом Choser.

## 🏁 Текущий статус (Февраль 2026)

### 1. Бэкенд (Cloudflare Workers/Pages)
*   **Модуляризация:** Монолит `src/index.js` успешно разделен на модули в папке `src/api/` (`auth.js`, `admin.js`, `tables.js`, `ai.js`, `mcp.js`).
*   **Синхронизация:** Код Worker (`src/`) и Pages Functions (`functions/`) теперь идентичен по логике, что обеспечивает работу API в обоих режимах.
*   **Удаление легаси:** Старый файл `src/html.js` удален. Проект полностью перешел на связку React (frontend) + Hono (backend).

### 2. Деплой (CI/CD)
*   **GitHub Actions:** Настроен деплой в Cloudflare Pages под именем **`choser-app`**. 
*   **Важно:** Для работы API в Pages нужно вручную привязать D1 database (имя `DB`) в настройках проекта в панели Cloudflare.

### 3. Автоматизация (Deep Research)
*   **Research Agent:** В папке `research_agent/` создан Python-агент (`agent.py`).
*   **Функционал:** Использует Gemini 2.0 Flash с Google Search Grounding для автономного исследования интернета. Сразу выдает структурированный JSON и умеет отправлять его напрямую в API Choser.

# Choser CF MVP

Cloudflare Workers + D1 + LangChain project for AI-driven wiki.

## API Endpoints

- `GET /` - Health check
- `GET /api/table/:id` - Get table data
- `POST /api/table` - Create table
- `POST /api/generate` - Generate table with Gemini

## MCP
- `GET /mcp/sse` - SSE endpoint for MCP
- `POST /mcp/messages` - Messages endpoint
